#  📱 CityConnect – Android App

🚀 CityConnect is an Android application that helps users discover the best local places in their city — from cozy coffee shops to restaurants, parks, shopping centers, and kid-friendly locations.

##  🌆 Features
🔍 Browse multiple categories:
- Coffee Shops ☕
- Restaurants 🍽️
- Parks 🌳
- Shopping Centers 🛍️
- Kid-friendly Places 🎡
## 📋 View curated listings with:
- Place name
- Short description
- Approximate cost
## 🎨 Clean and intuitive UI
- 📱 Smooth multi-screen navigation
- 👤 Beginner-friendly and easy-to-use design
## 🛠️ Tech Stack
- Language:Kotlin
- IDE: Android Studio
- UI Design: XML Layouts
- Architecture: Basic Activity-based navigation
